@startuml
actor Cliente
boundary PaginaWeb

Cliente -> PaginaWeb: Ingresar a la página
PaginaWeb --> Cliente: Muestra opciones de inicio (cliente / administrador)

Cliente -> PaginaWeb: Selecciona "Cliente"
PaginaWeb --> Cliente: Solicita nombre, apellido, teléfono y mail

Cliente -> PaginaWeb: Ingresa datos personales
PaginaWeb -> PaginaWeb: Validar datos

alt Datos válidos
    PaginaWeb --> Cliente: Acceso a interfaz principal del cliente
else Datos inválidos
    PaginaWeb --> Cliente: Mostrar mensaje "mail inválido" o "teléfono inexistente"
    Cliente -> PaginaWeb: Reintenta ingreso de datos
end
@enduml
