@startuml
actor Administrador
participant "Interfaz" as Web
participant "Controlador" as Controlador
participant "Servicio" as Servicio


Administrador -> Web: Ingresar a la página
activate Web

Web -> Administrador: Mostrar opciones (Cliente / Administrador)
Administrador -> Web: Seleccionar opción Administrador

Web -> Administrador: Mostrar opciones ("Agregar", "Eliminar", "Modificar")
Administrador -> Web: Seleccionar opción "Agregar"

Web -> Administrador: Mostrar opciones ("Agregar servicio", "Agregar profesional", "Agregar fecha disponible")
Administrador -> Web: Seleccionar "Agregar servicio"

Web -> Administrador: Solicitar datos del servicio (Servicio, Precio)
Administrador -> Web: Ingresar datos y presionar "Aceptar"

Web -> Controlador: crearServicio(tipo, precio)
activate Controlador
Controlador -> Servicio: << create >> Servicio(tipo, precio)
activate Servicio
Servicio --> Controlador: Servicio creado
deactivate Servicio
Controlador --> Web: Confirmación de carga
deactivate Controlador

alt Campos completos
    Web -> Administrador: Mostrar mensaje “Servicio agregado con éxito”
else Campo vacío
    Web -> Administrador: Mostrar mensaje “Hay un campo vacío, por favor rellénelo para continuar”
end

deactivate Web
@enduml
